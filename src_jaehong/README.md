# 인플루엔자 정보 포털

질병관리청 인플루엔자 정보 포털의 React 프론트엔드 애플리케이션입니다.

## 기능

- 📊 인플루엔자 신고 건수 및 유행률 통계 대시보드
- 📈 주간/월간/연간 추이 차트
- 🗺️ 지역별 통계 현황
- 👥 연령대별 통계 분석
- 📱 반응형 디자인

## 설치 및 실행

### 필수 요구사항

- Node.js 16.x 이상
- npm 또는 yarn

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm start
```

브라우저에서 [http://localhost:509](http://localhost:509)을 열어 확인하세요.

### 빌드

```bash
npm run build
```

프로덕션 빌드가 `build` 폴더에 생성됩니다.

## 사용된 기술

- React 18
- Material-UI (MUI) 5
- Recharts (차트 라이브러리)
- Axios (HTTP 클라이언트)

## 프로젝트 구조

```
src/
├── components/
│   ├── Header.js      # 헤더 컴포넌트
│   ├── Dashboard.js   # 메인 대시보드
│   └── Footer.js      # 푸터 컴포넌트
├── App.js             # 메인 앱 컴포넌트
├── App.css            # 앱 스타일
├── index.js           # 진입점
└── index.css          # 글로벌 스타일
```

## 주요 기능 설명

### 대시보드
- 총 신고 건수, 최근 주 신고 건수, 평균 유행률, 최근 주 유행률 등 주요 지표를 카드 형태로 표시
- 주간 인플루엔자 신고 건수 추이를 Area Chart로 시각화
- 연령대별 신고 건수를 Bar Chart로 표시
- 주간 유행률 추이를 Line Chart로 표현
- 지역별 통계를 테이블로 제공

### 필터링
- 기간 선택 (주간/월간/연간)
- 지역 선택 (전국/서울/부산/경기 등)
